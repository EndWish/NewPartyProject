### 프로젝트 소개  
```
친구나 지인들과 함께 즐기기 위한 게임을 만들기 위해 시작한 프로젝트.  
이전에 시도한 프로젝트의 실패를 발판삼아 새로이 도전.  
내가 좋아하는 게임들의 재미있는 요소들을 가져와 추가하고 수정하며, 게임 개발에 있어 여러가지 시도를 해볼 수 있는 프로젝트.  
  ```

### 게임에 대한 간단한 설명  
```
파티를 꾸려 던전을 공략하는 수집형 턴 제 RPG 게임. 친구와 함께 멀티플레이 가능.
```
